Installation
----------------------------------------------------------------------
Automatique via Module-Install (Admin/Gestion installation des modules)

Associé à Meta-lang et à infobulle :

 - Dans l'administration de npds_glossaire / liste des définitions - Vous n'avez ensuite plus qu'à cliquer sur "Meta" pour transférer
   votre terme, sa définition et le lien associé dans meta-lang !
 - Si vous éditez le terme : il faudra refaire le transfert vers meta-lang
 - Si vous supprimer le terme : il ne sera pas supprimé automatiquement de meta-lang (passer par l'admin de meta-lang pour ce faire)
